## ⚡ TL;DR - Super Quick Version

If you just want the commands:

```bash
# 1. Check your setup
bash scripts/check-setup.sh

# 2. Install dependencies
pip install -r requirements.txt
npm install -g serverless

# 3. Configure
cp .env.example .env
# Edit .env and change JWT_SECRET

# 4. Deploy
serverless deploy

# Done! Save the API endpoint URL
```
